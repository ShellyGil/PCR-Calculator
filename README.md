# PCR Genotyping Calculator

A Python tool (GUI + CLI) to calculate the master mix volumes for PCR genotyping. It supports variable sample numbers, safety excess percentages, and different Mix concentrations (2X or 5X).

## Features

* **GUI Mode:** Easy-to-use interface with sliders and buttons.
* **CLI Mode:** Command-line support for quick calculations or automation.
* **File Export:** Save your reaction recipes to a text file for lab records.
* **Flexible Inputs:** Adjust sample count, excess percentage, and mix concentration.
